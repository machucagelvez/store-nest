# Teslo API

1. Instalar o clonar proyecto
2. ```
   npm i
   ```
3. Clonar elarchivo `.env.template` y renombrarlo a `.env`
4. Ajustar la variables de entorno
5. Levantar la BD:

```
docker compose up -d
```

6. Ejecutar seed:

```
http://localhost:3000/api/seed
```

6. Iniciar el API:

```
npm ru start:dev
```
